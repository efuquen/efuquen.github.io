# Calculus

## Section 1: Limits

### Property 1.1: Limit of a constant function

If $F(x) = c$ for all $x$, where $c$ is some constant value, then

$\boxed{\lim\limits_{h \to a} F(h) = c}$

### Property 1.2: Limit of $x$

If $F(x) = x$, then

$\boxed{\lim\limits_{h \to a} F(h) = h}$
